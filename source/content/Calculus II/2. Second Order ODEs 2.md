### Euler type homogenous linear second order ODEs

$$
ax^2 \frac{d^2y}{dx^2} + bx \frac{dy}{dx} + cy = 0
$$

Euler equations can be solved by similar methods to homogenous linear equations with constant coefficients. Instead of using $y = e^{rx}$, we use $y = x^v$

If $y = x^v$:
$$
\begin{gather*}
y' = vx^{v-1} \\
y'' = v(v-1)x^{v-2}
\end{gather*}
$$
Substitute into $ax^2y'' + bxy' + cy = 0$"
$$
\begin{gather*}
ax^2[ν(ν−1)x^{ν−2}]+bx[νx^{ν−1}]+cx^ν = 0 \\
aν(ν−1)x^ν+bνx^ν+cx^ν=0 \\
x^ν(aν(ν−1)+bν+c)=0 \\
aν(ν−1)+bν+c = 0 \\
av^2 + (b - a)v + c = 0
\end{gather*}
$$
Which gives us the auxiliary equation: $av^2 + (b - a)v + c = 0$ where we solve for 2 distinct roots, $v_1$ and $v_2$.
